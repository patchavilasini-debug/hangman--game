# Hangman Game

A simple Hangman Game developed using programming concepts and logical problem-solving skills. The game allows users to guess letters and complete hidden words within a limited number of attempts. This project demonstrates string handling, loops, conditional statements, and user interaction.

## Features
- Interactive word guessing game
- Limited attempts for each round
- User-friendly interface
- Random word selection
- Score tracking

## Technologies Used
- Python

## Project Objective
The objective of this project is to develop an interactive word-guessing game that enhances logical thinking and programming skills. The game provides users with an engaging experience while demonstrating core programming concepts.

## How to Play
1. Start the game.
2. A hidden word will be selected randomly.
3. Guess one letter at a time.
4. Correct guesses reveal the letter in the word.
5. Incorrect guesses reduce the remaining attempts.
6. Guess the complete word before all attempts are used.
